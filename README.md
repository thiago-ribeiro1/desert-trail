# Desert Trail

Um jogo 2D simples onde o desafio é manter a moto equilibrada enquanto atravessa dunas no deserto.  
Apesar de ser um projeto simples 2D, reúne física básica, terreno procedural, sons (motor, clique, game over), tela de menu com música e uma interface responsiva.

Desenvolvido em HTML5 Canvas + JavaScript puro, como forma de praticar lógica, física 2D, manipulação de áudio e organização de código no front end.

## Funcionalidades

- Terreno gerado proceduralmente (dunas com variação suave)
- Física simples de inclinação, velocidade e colisão
- Moto com animação baseada no ângulo da pista
- Som do motor com volume dinâmico e sem cortes
- Som de clique, música de menu e efeito de game over
- Tela inicial com botão interativo

---

## Controles

| Tecla | Ação |
|-------|-------|
| **Seta para cima** | Acelera a moto |
| **Seta para baixo** | Ré / desacelera |
| **Setas esquerda / direita** | Inclina a moto |
| **R** | Reinicia a partida |
| **Clique no botão INICIAR** | Começa o jogo |
