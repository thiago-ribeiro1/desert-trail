# Desert Trail

Desert Trail é um mini-game 2D inspirado em clássicos como **Trial Bike**, onde você controla uma moto atravessando dunas no meio do deserto.  
O objetivo é simples: **acelerar, se equilibrar e sobreviver até a linha de chegada**. Cada subida, descida e curva da areia muda o ângulo da moto, exigindo reflexos rápidos para não virar de vez.

É um jogo feito para testar sua habilidade de equilíbrio e controle enquanto a física faz o resto.

Desenvolvido em HTML5 Canvas + JavaScript puro, como forma de praticar lógica, física 2D, manipulação de áudio e organização de código no front end.

## Funcionalidades

- Terreno procedural com dunas sempre diferentes  
- Física de inclinação e equilíbrio  
- Sons de motor, game over e trilha no menu  
- Tela inicial interativa  
- Experiência simples, leve e divertida 

---

## Controles

| Tecla | Ação |
|-------|-------|
| **Seta para cima** | Acelera a moto |
| **Seta para baixo** | Ré / desacelera |
| **Setas esquerda / direita** | Inclina a moto |
| **R** | Reinicia a partida |
